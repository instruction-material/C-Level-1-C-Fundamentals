# C Level 1 C Fundamentals

Starter and solution code for the C Level 1 C Fundamentals course.

This repo was split out of mixed C/C++ instructional material so the C foundations builds can live in a dedicated course repository instead of sharing `CPP-Level-1`.

The live Python to Java and C++ Bridge course links to one comparison build
from this repo for C/C++ workflow translation. That cross-link is intentional;
the repo itself remains the canonical home for the C foundations builds, not
for C++ Level 1 or C++ Level 3.
